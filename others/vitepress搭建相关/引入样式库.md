# 引入Element Plus

## 安装
我们需要安装Element Plus和Element Plus的图标库
这里作为生产依赖安装，使用的是pnpm
```bash
pnpm add element-plus @element-plus/icons-vue
```

# 引入Tailwind CSS

## 安装
我们需要安装Tailwind CSS和PostCSS和autoprefixer
这里使用pnpm作为开发依赖安装
```bash
pnpm add -D tailwindcss postcss autoprefixer
```
